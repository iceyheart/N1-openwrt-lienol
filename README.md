# 云编译 N1 OpenWrt 固件

**集成**
- SSR+、passwall、passwall2、nps、zerotier、kms、timectrol、KMS，其它服务略

**旁路网关模式**
- DHCP关、wireless关
- 默认IP 192.168.31.2 网关192.168.31.1，关闭桥接，开启IPV6，取消分配前缀
